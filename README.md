# 律思 · AI 法学知识库

一个纯前端的个人法学知识库，使用 React + Vite + TailwindCSS 构建。

## 功能

- 📚 知识条目管理（法条解释 / 法律概念 / 案例分析 / 考点总结 / 自由笔记）
- 🧠 法学⑩结构模板（法条→白话→考点→案例→争议→裁判→易错→真题→记忆→笔记）
- 🔍 全文搜索 + 学科分类筛选 + 标签系统
- 📱 响应式设计，手机可用
- 💾 数据全部存储在浏览器 IndexedDB 中（隐私安全）
- 🆓 零成本，GitHub Pages 免费托管

## 技术栈

- React 19 + Vite 8
- TailwindCSS 4
- Dexie.js (IndexedDB)
- React Router 7

## 本地运行

```
npm install
npm run dev
```

## 部署到 GitHub Pages

```
npm run deploy
```
